# GuliMail 分布式电商项目

本项目是一个基于 Spring Cloud Alibaba 的大型分布式电商系统。

## 核心模块
*   `gulimail-product`: 商品服务
*   `gulimail-order`: 订单服务
*   `gulimail-ai`: 智能客服服务 (New)
*   ...
