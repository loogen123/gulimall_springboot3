# 风险预警面板

本文件为手工维护，用于记录风险预警状态。

触发规则（手工判定）：
- 延迟预警：实际耗时 > 预计耗时 * 115%（或以当前完成百分比推算的应耗时 * 115%）
- 阻塞预警：阻塞=是，或出现技术/环境阻塞（依赖缺失、服务不可用、配置不可得）
- 逾期预警：当前日期 > 计划完成日期 且状态≠已完成
